# Blue Fair Golf Club

A modern Next.js website for **Blue Fair** — a premium golf club with a beautiful blue theme.

## Pages

- **Home** — Hero section, features, course preview, blog highlights, and CTA
- **About** — Club history, mission, values, timeline, and team
- **Blogs** — Golf tips, tournament news, and club updates
- **Contact** — Contact form, info cards, and FAQ

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Responsive design with mobile navigation

## Scripts

- `npm run dev` — Start development server
- `npm run build` — Build for production
- `npm run start` — Start production server
- `npm run lint` — Run ESLint
